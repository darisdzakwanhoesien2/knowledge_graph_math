# Derivation: Properties of the p-Norm

## 1. Goal
Prove that for \(p \ge 1\), the function  
\[
\|x\|_p = \left(\sum_{i=1}^n |x_i|^p\right)^{1/p}
\]
is a norm.

---

## 2. Positive Definiteness

Since each term \(|x_i|^p \ge 0\):

\[
\|x\|_p \ge 0.
\]

If \(x = 0\), all elements are zero so the norm is zero.  
Conversely,  
\[
\|x\|_p = 0 \Rightarrow |x_i|^p = 0 \ \forall i \Rightarrow x = 0.
\]

---

## 3. Homogeneity

\[
\|\alpha x\|_p = \left( \sum_{i=1}^n |\alpha x_i|^p \right)^{1/p}
= |\alpha| \left( \sum_{i=1}^n |x_i|^p \right)^{1/p}
= |\alpha| \|x\|_p.
\]

---

## 4. Triangle Inequality

Use **Minkowski’s inequality**:

\[
\|x + y\|_p \le \|x\|_p + \|y\|_p.
\]

Proof uses:

1. Hölder’s inequality  
2. Convexity of \(t^p\)

---

## 5. Conclusion

Thus \(\|\cdot\|_p\) satisfies all norm axioms.
